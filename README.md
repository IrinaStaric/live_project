# Live Project — Math Notes

A structured set of math notes (Linear Algebra, Calculus, Geometry) published as a clean documentation website with MkDocs and GitHub Pages.

## What’s inside
- Clear topic navigation (MkDocs Material)
- LaTeX-style math rendering
- Diagrams and visual intuition
- An interactive simulation page (sine wave + derivative/integral)

## Live site
The project is published with GitHub Pages.

## Tech stack
- MkDocs + Material theme
- Markdown + MathJax
- GitHub Pages

## Local run
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install mkdocs-material pymdown-extensions
mkdocs serve
